About Yuna API

Why Yuna?

I was inspired to develop Yuna because I realized that while the game is popular, there has been a huge lack of API that contains essential data that can be condensed into something simplistic for both developers and users alike.

Are the data from API updated?

As of writing, yes. Most of the data are referenced from the Genshin Wikia and if there is a major update, it shouldn't take more than a week post-update to be updated.

Can I contribute?

Yes! As of now, Yuna is open-source and as of now, I'm a solo developer, so I'd appreciate if someone is willing to work alongside.
