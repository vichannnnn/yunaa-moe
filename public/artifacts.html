Artifacts

Artifacts HTTP GET Request Method

Gets the data of all the available artifact sets.

GET https://yunaa.moe/api/artifacts/

Parameterized Artifact HTTP GET Request Method

Gets the data of a specific artifact set.

Currently, the supported parameters are:

Parameter Description
name Name of the artifact set.

Example Request

GET https://yunaa.moe/api/artifacts?name=Gambler

Example Response

{
"setName": "Gambler",
"artifactOrigin": "Gambler is an Artifact Set available at 3-star and 4-star rarities which can be obtained from The Lost Valley, Clear Pool and Mountain Cavern and Peak of Vindagnyr.",
"imageLink": "https://static.wikia.nocookie.net/gensin-impact/images/3/39/Item_Gambler%27s_Brooch.png/revision/latest?cb=20201120062914",
"wikiLink": "/wiki/Gambler",
"setEffect": {
"2 Set Effect": " Increases Elemental Skill DMG by 20%.",
"4 Set Effect": "Defeating an opponent has 100% chance to remove Elemental Skill CD. Can only occur once every 15s.\n"
}
}

Response Parameters

Type Description
setName Name of the artifact set.
artifactOrigin The known ways to obtain the artifact set.
imageLink The image of the artifact set, specifically only the flower part.
wikiLink The Genshin Wikia link of the artifact set.
setEffect Shows a key-value pair of 2 Set Effect and 4 Set Effect
2 Set Effect The set effect description of the 2-Set Effect.
4 Set Effect The set effect description of the 4-Set Effect.
