Characters

Character HTTP GET Request Method

Gets the data of all the available playable characters.

GET https://yunaa.moe/api/characters/

Parameterized Characters HTTP GET Request Method

Gets the data of a specific playable character.

Currently, the supported parameters are:

Parameter Description
name Name of the character.

Example Request

GET https://yunaa.moe/api/characters?name=Hu_Tao

Response Parameters

Type Description
characterName Name of the character.
wikiLink The Genshin Wikia link of the character.
imageLink The image of the character.
rarityLevel The rarity star of the character.
weaponType The type of weapon that the character wields.
characterVision The vision of the character.
secondaryStats The secondary attribute of the character apart from the default ATK.
characterStats A list of the detailed scaling range for every level tier and their respective base ATK, DEF, HP secondary stats from level 1 to 90.
