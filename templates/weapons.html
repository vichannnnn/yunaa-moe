Weapons

Weapon HTTP GET Request Method

Gets the data of all the available weapons.

GET https://yunaa.moe/api/weapons/

Parameterized Weapon HTTP GET Request Method

Gets the data of a specific weapon.

Currently, the supported parameters are:

Parameter Description
name Name of the weapon.
type The weapon type e.g. Claymores

Example Request

GET https://yunaa.moe/api/weapons?name=Staff_of_Homa

Response Parameters

Type Description
weaponName Name of the weapon.
weaponType Type of the weapon.
secondaryStats The secondary attribute of the weapon apart from the default ATK.
baseAttackRange The scaling range of the base attack from level 1 to 90.
secondaryRange The scaling range of secondaryStats from level 1 to 90.
weaponStats A list of the detailed scaling range for every level tier and their respective base attack and secondary stats from level 1 to 90.
effectName Title name of the weapon's effectDescription.
effectDescription Description of the weapon's effect.
imageLink Image link of the weapon.
